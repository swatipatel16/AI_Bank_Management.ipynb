 AI-Enhanced Bank Management System

 Overview
A Python-based bank management system integrated with a machine learning model to detect fraudulent transactions.  
This project demonstrates how AI can enhance the security of digital banking operations.

---

Features
- User authentication (login/logout)
- Deposit, withdraw, and transfer money
- Fraud detection using Random Forest Classifier
- Interactive console interface

---

 Technologies Used
- Python  
- Pandas, Scikit-learn  
- Google Colab

---

 How to Run
1. Open the `.ipynb` file in Google Colab  
2. Run all cells sequentially  
3. Login with sample users (like `swati` / password `1234`)  
4. Test deposits, withdrawals, and transfers  

---

 Future Enhancements
